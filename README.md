## 游戏规则
### 方块生成规则

与标准俄罗斯方块一致，一次性生成一组 7 个方块并打乱次序。用完后再生成下一组。

### 计分规则

* 每消去 1 行计 1 分
* 同时消去 3 行，计 4 分
* 同时消去 4 行，计 6 分
